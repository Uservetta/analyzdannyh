# Исследование подходов к автоматизации планирования промышленных территорий

Репозиторий для научно исследовательской работы (НИР).

> **Автор:** Солодова Елизавета  
> **Тип работы:** НИР  
> **Статус:** В процессе

## Цель работы

Разработать методику автоматизированного планирования промышленных территорий с использованием современных методов анализа данных.

## Задачи исследования

1. Анализ существующих методов планирования промышленных территорий
2. Изучение программных инструментов для автоматизации планирования
3. Разработка прототипа системы поддержки принятия решений
4. Тестирование на модельных данных
